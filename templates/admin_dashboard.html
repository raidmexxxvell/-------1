Сезон

                    

                        
                        
                        
                        
                        
                        

                            
                            
                            
                        

                    

                    
План — пробный запуск без изменений. Мягкий — без очистки старых данных. Полный — удаляет старые служебные данные. Deep + Полный — дополнительно удаляет матчи прошлого турнира и импортирует новое расписание.

                    

                    

                        Подсказки по ошибкам отката:
                        
    
                            
  • Not Found — эндпоинт недоступен. Используется встроенный путь POST /api/admin/season/rollback; если видите это, обновите сервер до последней версии.
    • 
                            
  • no_rollover_history — нет записей в журнале season_rollovers. Сначала выполните «Полный сброс» (rollover).
    • 
                            
  • active_mismatch — активный турнир отличается от последнего rollover. Можно выполнить с параметром ?force=1 (кнопка попросит подтверждение).
    • 
                            
  • tournament_not_found — записи турниров не найдены по id. Проверьте целостность данных БД.
    •
